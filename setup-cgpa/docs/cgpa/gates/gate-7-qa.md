# Gate 7 — QA & recette

> Frontière entre la phase **09 (QA & recette)** et la phase **10 (Production)**.

## Conditions d'entrée
- Gate 6 statué Go.
- Incrément développé et déployé en `staging`.
- Code audité (prompt 08).

## Livrables obligatoires
- [ ] `templates/rapport-qa.md` complété.
- [ ] `templates/pv-recette.md` signé.
- [ ] `checklists/checklist-qa.md` renseignée.

## Critères de validation
- [ ] Couverture des critères d'acceptation tracée (CDC → cas de test).
- [ ] Tests unitaires, intégration, e2e et non-régression passants.
- [ ] Exigences non fonctionnelles évaluées (sécurité, perf).
- [ ] Aucune anomalie bloquante ouverte.
- [ ] PV de recette prononcé.
- [ ] Score ≥ 14/20 recommandé.

## Risques bloquants
- Anomalies bloquantes non traitées.
- Couverture insuffisante / non tracée.
- Sécurité ou performance non validées.
