# Gate 0 — Idée & opportunité

> Frontière entre la phase **01 (Idée)** et la phase **02 (Expression du besoin)**.

## Conditions d'entrée
- Une idée est formulée et portée par un sponsor.
- La stack cible et les contraintes principales sont connues.

## Livrables obligatoires
- [ ] `templates/fiche-idee.md` complété.

## Critères de validation
- [ ] Problème et valeur clairement énoncés.
- [ ] Cible / personas identifiés.
- [ ] Hypothèses clés rendues testables.
- [ ] Risques majeurs identifiés.
- [ ] Score d'opportunité ≥ 8/20.

## Risques bloquants
- Valeur ou cible non démontrée.
- Idée non différenciée / déjà couverte sans avantage.
- Contraintes rédhibitoires (compétences, budget, temps).
