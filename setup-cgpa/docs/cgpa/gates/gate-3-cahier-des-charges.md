# Gate 3 — Cahier des charges (CDC)

> Frontière entre la phase **04 (CDC)** et la phase **05 (Architecture)**.
> ⚠️ Fait partie du **verrou de codage**.

## Conditions d'entrée
- Gate 2 statué Go (ou Go sous réserve).
- Étude de faisabilité validée.

## Livrables obligatoires
- [ ] `templates/cahier-des-charges.md` complété.

## Critères de validation
- [ ] Exigences fonctionnelles détaillées et priorisées.
- [ ] Exigences non fonctionnelles spécifiées (sécurité, perf, dispo).
- [ ] Critères d'acceptation présents pour chaque exigence clé.
- [ ] Interfaces / intégrations spécifiées.
- [ ] Matrice de traçabilité besoin → exigence.
- [ ] Score ≥ 8/20.

## Risques bloquants
- Exigences non testables ou sans critère d'acceptation.
- Non fonctionnel / sécurité insuffisamment spécifiés.
- Traçabilité absente.
